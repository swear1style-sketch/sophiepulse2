# SophiePulse2

Static marketing landing page for SofiaPulse.

## Deployment

- This is a static HTML/CSS/JS site.
- The root file is `index.html`.
- Vercel will deploy automatically when this repository is connected.

## Notes

- Video files in `assets/videos/` are tracked with Git LFS.
- If you need to push changes from another machine, install Git LFS first:

```bash
git lfs install
```
